🎧 Project Overview
M3-Hackathon-2021 is a high-performance, minimalist web application built to serve as a 24/7 continuous audio/visual broadcast hub for M3 MUSIC ARTIST.

The philosophy behind this site is "Pure Audio Retention." By stripping away the clutter of traditional artist websites, this platform focuses on a single, uninterrupted loop of the artist’s music, paired with high-conversion links to purchase and stream on external platforms.

🚀 Core Features
Infinite Loop Engine: Utilizes a specialized API implementation to ensure the music plays nonstop, 24 hours a day, without manual refreshes.

Direct-to-Fan Ecosystem: A centralized "Command Center" providing instant access to:

Digital Purchase Storefronts

Streaming Platforms (Spotify, Apple Music, Tidal)

Social Media Portals

Performance Optimized: Extremely lightweight code for near-instant load times on mobile and desktop.

Visual Immersion: A full-screen video background layer that syncs with the audio for a cinematic user experience.

🛠️ Technical Stack
HTML5/CSS3: Used for the semantic structure and responsive, high-contrast UI.

JavaScript: Logic for handling browser autoplay policies and the looping playback engine.

YouTube iFrame API: Integrated for high-fidelity audio streaming and backend video hosting.

GitHub Pages: Deployed as a static site for maximum uptime and reliability.

⚡ Setup & Customization
To update the music or links, follow these steps:

Change the Song: Open index.html and replace the VIDEO_ID in the src attribute with your new YouTube Video ID.

Update Links: Navigate to the <div class="links"> section and swap the href values with the artist's current Spotify/Apple Music URLs.

Push to Live: Commit your changes to the main branch. GitHub Pages will automatically update the live site at:

https://skrs-ai.github.io/M3-Hackathon-2021/

🏆 Hackathon Context
This project was conceptualized and deployed during the 2021 M3 Hackathon. The goal was to solve the "Paradox of Choice" in music marketing by providing fans with a single, unignorable path to the music.
